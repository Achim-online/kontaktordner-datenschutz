## 1. Datenspeicherung
Alle in der App eingegebenen Daten (Kontakte, Termine, Notizen) werden 
ausschließlich lokal auf dem Gerät des Nutzers gespeichert. 
Der Entwickler hat keinen Zugriff auf diese Daten.

## 2. iCloud-Synchronisation
Sofern der Nutzer iCloud aktiviert hat, werden Daten verschlüsselt in 
seinem persönlichen Apple iCloud-Account gespeichert. Es gelten die 
Datenschutzbestimmungen von Apple Inc. (apple.com/de/privacy).

## 3. Push-Benachrichtigungen
Die App sendet lokale Benachrichtigungen direkt auf dem Gerät des Nutzers. 
Es werden keine Daten an externe Server übertragen.

## 4. Zugriff auf iPhone-Kontakte
Die App kann mit Zustimmung des Nutzers auf die iPhone-Kontakte zugreifen, 
um Daten zu importieren. Dieser Zugriff erfolgt nur auf ausdrücklichen 
Wunsch des Nutzers.
